## COMPOSITION STANDARDS

### Harmonic richness
- Use full chord voicings (3–4 simultaneous notes in chordal passages)
- Add 7ths, 9ths, suspensions for colour
- Plan a real chord progression — don't just repeat I-IV-V
- Use secondary dominants, borrowed chords, or modal mixture
- Voice lead smoothly: inner voices move by step or stay still

### Melodic quality
- Write a singable melody: clear shape — rise, climax, descent
- Mix steps (2nds) and leaps (3rds, 4ths, 6ths)
- Avoid too many repeated notes
- Rhythmic variation — not just quarter notes
- Use motifs that return and develop (invert, augment, sequence)

### Rhythmic interest
- Mix note lengths: quarters, eighths, halves, dotted rhythms
- Syncopation: notes starting on offbeats (s=2,6,10,14)
- Rhythmic contrast between voices
- Avoid all voices moving at the same rhythm simultaneously

### Structure
- Shape the piece: start moderate, build to climax, resolve at end
- Leave occasional rests in the melody for breathing room
- Strong opening bar 1, satisfying resolution in final bar

### Voice independence
- Bass: roots and fifths, quarter/half notes, low register (C2–B3)
- Inner harmony: chords, mid register (C3–B4)
- Melody: singable line, upper register (C4–B6)
- Avoid parallel motion between voices

## ARTICULATION — USE c AND o FIELDS FOR EXPRESSION
These two optional fields are your expressive tools — use them deliberately:

### c = end_cutoff_percent (staccato/legato control)
Omit c for legato (note rings to near-full duration — default, smooth, lyrical).
Add c for shorter articulations:
  c:75  → slightly detached (walking bass, inner chord notes)
  c:60  → marcato/portato (melody accents, punchy inner voices)
  c:45  → staccato (light, bouncy passages)
  c:30  → very staccato (pizzicato-style, percussive attacks)
  c:50 with d:0 → grace note / ghost note (very short ornament)

Use legato (omit c) for: melody long tones, sustained harmonies, bass held notes
Use c:50–75 for: walking bass lines, inner chord notes, detached melody phrases
Mix both within a passage for natural breathing and phrasing

### o = offset_percent (micro-timing / swing / groove)
Omit o for straight eighth-note grid placement.
Use o for rhythmic feel:
  o:15–20  → subtle swing or humanization on any note
  o:25–33  → jazz/swing feel on eighth notes
  o:50     → full swing / triplet-feel eighths
Add o only where you want the groove or a slight push/pull — not on every note.

### Example — lyrical melody phrase:
  Bar 5: { p:"E5", s:0, d:6 }           ← long tone, legato, no c
          { p:"D5", s:6, d:2, c:70 }     ← passing note, slightly detached
          { p:"C5", s:8, d:6 }           ← arrival, legato
          { p:"B4", s:14, d:2, c:60 }    ← pickup, short

### Example — walking bass:
