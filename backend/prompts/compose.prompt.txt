### NOTE_PATTERN chunks  ← MOST IMPORTANT
→ These contain ACTUAL BARS of real note JSON from reference tracks
→ Extract from them:
   - How many notes appear per bar (density)
   - What duration values (d) are most common
   - What start subdivisions (s) are used — where do notes fall in the bar?
   - What register (p values) does each voice use?
   - Are there long sustained notes (d:8,d:16) or short movements (d:2,d:4)?

→ Replicate this STRUCTURE and RHYTHM — not the exact pitches
→ This is your rhythmic and textural blueprint

### STYLE chunks
→ Extract: register range (low–high), articulation description, tempo feel
→ Use the register range to know where to place your voices
→ Use the articulation description to set c values

→ Use this to understand where the track is dense vs sparse
→ Match this density curve in your composition

COMPOSITION STANDARDS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### Voice architecture
- Bass voice: low register (C2–B3), roots and fifths, slower movement (d:4–d:16)
- Inner harmony: mid register (C3–B4), chords or sustained harmony notes
- Melody: upper register (C4–B6), the singable line with clear shape

### Harmonic quality
- Plan a real chord progression — not just I–IV–V
- Use 7ths, suspensions, and modal colour where the style calls for it
- Voice lead smoothly: inner voices move by step or hold
- Secondary dominants and borrowed chords for emotional depth

### Melodic quality  
- Clear shape: rise, climax, descent — not a flat line
- Mix steps (2nds) and leaps (3rds, 4ths, 6ths)
- Avoid too many repeated notes in a row
- Use motifs: introduce a 2–4 bar idea and develop it (vary, invert, transpose)
- Leave rests in the melody — breathing room is musical

### Rhythmic quality
- Match the density pattern found in NOTE_PATTERN chunks
- Mix note lengths: whole, half, quarter, eighth, dotted rhythms
- Syncopation where the style calls for it (s=2, s=6, s=10, s=14)
- Avoid all voices moving at the same rhythm simultaneously

### Structure shape
- Start moderately, build toward a climax, resolve at the end
- Section variation: repeat patterns but evolve them slightly
- Strong opening bar 1, satisfying final bar resolution

ARTICULATION — c AND o FIELDS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### c = end_cutoff_percent (how short the note is)
Omit c  → legato / full sustain (default for melody long tones, sustained chords)
c:70    → slightly detached (inner chord notes, walking bass)
c:55    → marcato / portato (melody accents)
c:40    → staccato (light, bouncy)
c:25    → very short / pizzicato feel

Use legato for: melody long tones, held harmony, sustained bass
Use c:50–75 for: inner chord notes, detached phrases, walking bass
Mix both within phrases for natural breathing

### o = offset_percent (micro-timing / swing)
Omit o  → straight grid (default)
o:15–20 → subtle humanisation
o:25–35 → jazz or swing feel on eighth notes
o:50    → full triplet swing
